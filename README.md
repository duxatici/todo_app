# TODO App

FastAPI приложение для управления задачами.

## Возможности

- Добавление задач
- Просмотр списка задач
- Отметить задачу выполненной

## Структура

```
todo_app/
├── models/       # Модели данных
├── repositories/ # Работа с данными
├── services/    # Бизнес-логика
└── main.py     # Точка входа
```
